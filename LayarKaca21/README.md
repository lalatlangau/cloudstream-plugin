# LayarKaca21 Plugin untuk CloudStream

Plugin ini menambahkan dukungan untuk LayarKaca21 ke aplikasi CloudStream.

## Fitur
- Streaming film dan serial dari LayarKaca21
- Pencarian konten
- Multiple video quality
- Subtitle support

## Instalasi
1. Buka aplikasi CloudStream
2. Pergi ke Settings > Extensions
3. Tambahkan repository URL
4. Cari dan instal plugin "LayarKaca21"

## Pembangunan
Plugin ini dibangun menggunakan:
- Kotlin
- CloudStream3 SDK
